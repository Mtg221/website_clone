# Website Clone

Clone d'un site web vitrine avec galerie portfolio et navigation.

**Stack :** HTML · CSS · JavaScript Vanilla

---

## Structure

```
website_clone/
├── index.html
├── styles.css
├── script.js
└── images/
    ├── accueil/    ← Images de la page d'accueil
    └── portfolio/  ← Photos paysages et portraits
```

---

## Lancer le projet

Ouvre `index.html` dans un navigateur — aucune installation requise.

---

## Objectif pédagogique

- Reproduire la mise en page d'un site existant
- Pratiquer HTML/CSS avancé (galerie, navigation)
- Intégrer des médias et images
